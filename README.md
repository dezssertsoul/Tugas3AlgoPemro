Aplikasi konsol sederhana menggunakan Java untuk menghitung total gaji karyawan berdasarkan golongan (A, B, C) dan jumlah jam lembur yang dilakukan.

Fitur Utama:

Perhitungan Gaji Pokok: Menetapkan gaji pokok berdasarkan golongan karyawan.

Perhitungan Lembur: Menghitung gaji lembur berdasarkan persentase yang disesuaikan dengan jumlah jam lembur.

Output Rincian: Menampilkan detail gaji pokok, gaji lembur, dan total gaji yang diterima.


Persyaratan Sistem:

Untuk menjalankan program ini, memerlukan:

Java Development Kit (JDK): Versi 8 atau yang lebih baru.

Lingkungan Pengembangan (IDE) Java: Seperti NetBeans, IntelliJ IDEA, atau Eclipse (disini saya pakai Netbeans)
